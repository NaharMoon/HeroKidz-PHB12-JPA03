Hero Kidz - Next.js E-commerce Project

# Hero Kidz

A modern Next.js based toy e-commerce web application for children's products.

## Live Site
https://hero-kidz-phb12-jpa03.vercel.app

## GitHub Repository
https://github.com/NaharMoon/HeroKidz-PHB12-JPA03

## Features
- Responsive homepage
- Product showcase page
- Blog page
- Authentication setup
- Modern UI design
- Vercel deployment

## Tech Stack
- Next.js
- React
- Tailwind CSS
- DaisyUI
- MongoDB
- NextAuth

## Notes
This project is deployed on Vercel. Some dynamic product data depends on database configuration.

## Run Locally

```bash
npm install
npm run dev
